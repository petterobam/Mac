# Mac 初始化教程

> Mac配置相关，一键装机或系统恢复使用。

1. 格式化和系统重装
    - https://support.apple.com/zh-cn/HT204904

2. 系统环境初始化
    ```
    git clone https://github.com/petterobam/Mac.git
    cd Mac
    sh init.sh
    ————————————————————————
    以上命令将让你选择安装：
    # homebrew
    # zsh
    # vim
    # jdk
    # maven
    # ruby 相关环境
    # node 
    # python3
    ```
3. 安装常用软件
    - Chrome
    - 微信
    - 网易云
    - WPS
    - 网易邮箱大师
4. 安装强化功能插件
    - https://macwk.com/soft/path-finder
    - https://macwk.com/soft/yesplaymusic
    - https://www.macwk.com/special
